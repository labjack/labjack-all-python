labjack: Package that installs all LabJack Python wrappers
07/06/2022
support@labjack.com


This package is essentially an alias to install labjack-ljm (for
T-series devices) and LabJackPython (for UD devices). We recommend
downloading the individual package for your device rather than this
package.

See our Python documentation:

* T-series documentation:
  https://labjack.com/support/software/examples/ljm/python
* UD documentation:
  https://labjack.com/support/software/examples/ud/labjackpython


Installation Requirements:

* Windows, Mac OS X / macOS, or Linux
* LJM library: https://labjack.com/support/software/installers/ljm
  ~and/or~
  UD library or Exodriver (for UD devices):
  https://labjack.com/support/software/installers/ud
  https://labjack.com/support/software/installers/exodriver
* Python 2.7 or 3.x.
